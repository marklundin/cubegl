Cuber

This is an WebGL version of Cuber, an interactive programmable Rubiks cube. It was built using and three.js uses a lot of post processing taken from Wagner.js.

Cuber has a modular backend the allows various rendering output such as css, svg, canvas, or - as you can see here - WebGL

Cuber and CubeGL was built by mark lundin with some friends at Google.
